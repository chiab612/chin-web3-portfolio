# 🧡 Solana Pay 公益捐款構想

## 應用情境
- 台灣學生社團、NGO 想收捐款，但不會做網站、也怕被詐騙利用
- 透過 QR Code + Phantom 錢包就能捐款，系統自動記錄金額、時間

## 技術構想
- Solana Pay (產生付款用的 QR)
- Helius API (監控鏈上交易)
- Stripe USDC 整合（讓沒有錢包的人也能刷卡）

## 預期特色
- 無需註冊，透明記錄
- 可嵌入 Linktree 或官網
- 捐款排行榜、留言板

## 適合族群
- Web3 公益組織、學生會
- 需要收費的線下講座活動

## 補充說明
- 即使我不會寫程式，我也能構想整個系統流程
- 我希望未來與工程師合作實作這個原型
